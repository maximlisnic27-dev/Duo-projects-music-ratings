# 🎵 PublicOpinion

**PublicOpinion** este un website modern care permite utilizatorilor să caute melodii și să vadă ratingul acestora în timp real.

## 📌 Descriere

Proiectul oferă o soluție simplă pentru a descoperi rapid informații despre muzică. Utilizatorul introduce numele unei melodii sau al unui artist, iar aplicația afișează automat date relevante precum ratingul, titlul și imaginea albumului.

## 🚀 Funcționalități

* 🔍 Căutare melodii și artiști
* ⭐ Afișarea ratingului
* 🎤 Informații despre melodie și artist
* 🖼️ Cover album
* ⚡ Răspuns rapid prin API

## 🛠️ Tehnologii folosite

* HTML
* CSS
* JavaScript
* API muzică (ex: Spotify / Last.fm)

## 🎯 Scopul proiectului

Scopul proiectului este de a oferi utilizatorilor o platformă simplă și rapidă pentru a verifica popularitatea și ratingul muzicii.

## 🔮 Dezvoltări viitoare

* Sistem de conturi
* Comentarii și ratinguri de la utilizatori
* Liste de favorite
* Topuri muzicale
* Filtrare avansată
